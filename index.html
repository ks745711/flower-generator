Flower Generator

Controls

  • Show introduction: show or hide this introduction
  • Flower petals: number of petals on each flower
  • Flower shape: width of each flower petal
  • Flower redness: red value in the flowers' RGBA color
  • Flower greenness: green value in the flowers' RGBA color
  • Flower blueness: blue value in the flowers' RGBA color
  • Flower transparency: alpha transparency value in the flowers' RGBA color
  • Text: Custom text on top of the flowers
  • Text size: The size of custom text
  • Text color: The color of the custom text
  • Star color: The color of the star
  • BG color: The color of the background
  • Dowload png: Download a png image of your custom creation
